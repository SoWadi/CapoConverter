# ⫷ CapoConverter ⫸

➠ A tool for musicians.

# Digital capodastre 🎸 🎻 🎤
Converts chords to adapt the music instruments to the singer's voice tune. 

# Random tunes 🎲 🎼
Generates randomly progression of chords to ease the initiation of a composition and/or a jam session.
